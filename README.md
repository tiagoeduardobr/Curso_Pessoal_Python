# Portfólio de Estudos em Python

Bem-vindo ao meu portfólio de estudos em Python! Este repositório serve como um registro do meu progresso e aprendizado na linguagem Python e em tecnologias de desenvolvimento de software.

## Sobre Este Repositório

Aqui eu documento os projetos, exercícios e conceitos que estudo, aplicando as melhores práticas de engenharia de software, como versionamento com Git, documentação e testes.
A metodologia é incremental, começando pelos fundamentos e avançando para tópicos avançados como APIs e IA generativa. Cada aula inclui código comentado, seguindo PEP 8 e "The Zen of Python".

## Aulas e Conceitos Estudados

Aqui estão as aulas desenvolvidas até agora, organizadas em módulos sequenciais. Cada uma é um mini-projeto com código executável e comentários didáticos.

*Todas as aulas foram padronizadas com comentários didáticos, docstrings e exercícios executáveis para melhor aprendizado.*
*Próximas aulas em desenvolvimento: Entrada e Saída, Comentários e Indentação, etc.*

## Tecnologias Utilizadas

* **Linguagem:** Python 3.x
* **Versionamento:** Git e GitHub (com workflows de branches e PRs)
* **Boas Práticas:** PEP 8, docstrings, e princípios de "The Zen of Python"
* **Futuras Integrações:** FastAPI, SQLAlchemy, e IA generativa (ex: Google Generative AI)

## Como Executar

1. Clone o repositório: `git clone https://github.com/tiagoeduardobr/Curso_Pessoal_Python.git`
2. Navegue para uma aula: `cd <numero>-<topico>` (ex: `cd 01-hello-world`)
3. Execute o código: `python main.py`
4. Para aulas futuras com dependências, instale via `pip install -r requirements.txt` (quando aplicável).

## Como Contribuir

1. Faça um fork do repositório.
2. Crie uma branch para sua contribuição: `git checkout -b feat/nova-aula`.
3. Faça commits semânticos e abra um Pull Request.
4. Sugestões: Adicione exercícios, melhore comentários ou proponha novas aulas.
